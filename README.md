This project applies multiple machine learning algorithms—SVM, Logistic Regression, Random Forest, and Naive Bayes- to classify tumors as malignant or benign using the Breast Cancer dataset.

Objectives: 
Train and compare different classification models
Evaluate performance using confusion matrices
Visualize results and feature importance
Apply PCA for dimensionality reduction

Models Used: 
Support Vector Machine (SVM)
Logistic Regression (LR)
Random Forest (RF)
Naive Bayes (NB)

Evaluation: 
Accuracy, Precision, Recall, F1-score
Confusion Matrix for each model
ROC curves (optional)

Visualizations: 
Correlation heatmap
Feature importance (RF)
Confusion matrix plots

PCA: 
Principal Component Analysis (PCA) is used to reduce dimensionality and visualize data in 2D while retaining maximum variance.

Dataset: 
Breast Cancer dataset (e.g., from sklearn or UCI repository)

Outcome: 
Compare model performance
Identify the best-performing classifier
Understand data structure through PCA and plots
